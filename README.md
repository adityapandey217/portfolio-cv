My portfolio using django.

# Local Setup

```json
git clone https://github.com/shaishguni/portfolio.git
```
```json
virtualenv venv && source venv/bin/activate
```
```json
pip install Django
```
```json
cd portflio
```
```json

python manage.py migrate
```
```json
python manage.py runserver
```


After that your are good to go.
